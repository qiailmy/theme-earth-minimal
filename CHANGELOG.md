# 更新记录

## 1.1.1

- 修复手机和平板通过 PJAX 切换页面后汉堡菜单不会自动收起的问题
- 更新 PJAX 资源版本参数，避免浏览器继续使用旧缓存

## 1.1.0

- 增加渐进增强式 PJAX 站内导航，减少连续浏览时的整页刷新
- 支持首页、文章、页面、分类、标签、归档、作者、友链、图库和瞬间之间快速切换
- 同步页面标题、URL、历史记录和目标页样式，并重新运行页面入口模块
- 增加轻量顶部加载反馈，请求或页面结构异常时自动回退完整刷新
- 外链、下载、新窗口链接和同页锚点保持浏览器默认行为

## 1.0.4

- 完全移除移动菜单的透明全屏覆盖层
- 将菜单按钮触摸区域从 18px 扩大到 40px
- 为主样式增加版本参数，强制手机浏览器刷新旧缓存

## 1.0.3

- 修复紧凑菜单点击后立即关闭、表现为无法展开的问题
- 阻止菜单按钮点击冒泡，并禁止透明背景层拦截操作

## 1.0.2

- 将手机和平板的展开菜单缩小为右上角紧凑面板
- 菜单宽度不超过 280px、高度不超过视口 65%，超出内容内部滚动
- 移除菜单展开时的大面积半透明模糊遮罩

## 1.0.1

- 桌面端合并主菜单和分类筛选，移除重复项目
- 固定顺序为：首页、默认分类、壁纸、日常、测试、医学、瞬间、友联
- 保留移动端折叠菜单和分类筛选

## 1.0.0

- 从官方 Earth 1.16.1 发布包重新建立干净基线
- 改为纯白、非吸顶导航和低装饰的阅读优先界面
- 默认关闭首页大图模块和文章顶部封面，默认使用单列文章列表
- 优化列表封面加载优先级、头像懒加载和图片异步解码
- 延迟渲染非首屏文章及侧栏组件，缩短交互动效
- 自定义 CSS 合并到主样式，不增加请求
